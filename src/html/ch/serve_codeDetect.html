原始碼檢測

產品諮詢

近年資安攻擊頻起，駭客經常利用程式開發的漏洞或弱點進行攻擊與資料竊取，為確保程式碼的品質和安全性，金管會及公務機關釋出相關法規，要求指定特定企業/上市櫃公司需執行源碼檢測，找出潛在風險，確保使用者的資料安全。

為協助企業解決相關問題及遵循政府資安法的規定，智慧資安科技遵循國際標準規範，提供「靜態原始碼檢測分析工具」，採用「OpenText Fortify Static Code Anaylzer」，檢視原始程式碼的方式，尋找並指出程式中潛藏的安全性弱點，分析其弱點種類、攻擊路徑等檢測分析報告，讓開發人員快速知道結果，以進行程式修改，避免弱點產生。
