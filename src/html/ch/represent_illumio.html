Illumio Zero Trust Segmentation 零信任微分隔解決方案 主要優勢

Illumio 通過構建獨特的自適應安全架構，可自內而外安全監控和防護每個工作單元或應用，而非基於如防火牆等傳統的邊界防護。Illumio 可協助不同企業部署 Micro-Segmentation 及 Policy Automation，並將零信任架構的部署簡單化。

Illumio 的 Micro-Segmentation（微切分）主要目的為

  1. 限制外圍突破的網路東西向傳播及感染
  2. 支持多樣化技術部署
  3. 獲得應用的可視化能力
  4. 利用已經擁有的東西
  5. 自動化和编排化 IT 維運和資安流程

Illumio 零信任解決方案提供三種產品線

  1. Workload SecurityIllumio Core
    是一種簡單、快速且經過驗證的微分段解決方案，通過提供智慧可視性、極其簡單的策略建立引擎以及自動分段和執行來阻止攻擊移動。
  2. Cloud SecurityIllumio CloudSecure
    協助企業獲得無與倫比的洞察力、優先考慮工作並為任何雲應用程序實施安全策略。Agentless 的實時可見性，可幫助企業瞭解安全風險並跨混合雲和多雲協調雲工作負載策略。
  3. Endpoint Security
    當所有員工都擁有一台筆記型電腦時，只要一台筆記型電腦被入侵，單個端點的勒索軟體或惡意軟體攻擊很容易立即橫向移動到數百或數千個端點。Illumio Endpoint 是將勒索軟體和惡意軟體包含到單個端點的唯一解法 – 即使尚未檢測到。Endpoint Security 將成為端點安全力量倍增器，為端點提供零信任優勢！

※ 適用產業：金融、高科技製造、政府、醫療保健、航空運輸、零售業等
