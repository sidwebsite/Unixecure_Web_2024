Rapid7 情報戰總司令 Threat Command 外部威脅 Take it Down!

2024 / 04 / 16
  • Rapid7

難以觸及的外部威脅

現今的網路世界，到處都有潛藏的威脅資訊，除了公開的明網 Clear Web 與表網 Surface Web，還有難以捉摸的地下網路環境，如：深網 Deep Web 與暗網 Dark Web。若您希望儘早掌握威脅情資， 得知自身企業是否成為駭客攻擊的目標，則勢必需要在網路世界裡追蹤您的數位足跡 Digital Footprint。正因如此，許多資安分析專家會嘗試潛伏至這些無法直接存取的網路環境中，從中觀察駭客之間的互動， 打探各種潛藏的非法攻擊與個資交易行為。

識別、評估、緩解

Rapid7 可以根據企業的數位足跡，採取自動分析與偵測查詢，透過簡易的步驟進行外部威脅緩解，將情報與企業資產進行關聯性分析， 使您面對企業、員工、source code、品牌與客戶的外部威脅時，能夠主動識別、評估，並且修復，此舉對於有效的威脅防護至關重要。 Rapid7 與市場上或傳統的其他威脅情資解決方案不同，為您省去了繁重的負擔及部署，有效降低威脅情報的複雜性，擁有威脅情資即時性的優勢。 透過 Rapid7 的威脅情資分析，協助您針對即將發生的攻擊提供事先預警，為企業超前部署！

All-In-One 威脅情資解決方案

Rapid7 Threat Command 可將外部情報對應到您獨特的數位足跡（或稱：數位資產），能夠持續偵測針對您業務上的關鍵威脅，可從明網、深網及暗網中， 提供量身定制的資安情報，依據嚴重性提供不同類別的安全警報，其中包含風險類型（例如：網絡釣魚、品牌安全、資料外洩等）及外部來源（例如：黑客論壇、社交媒體、黑市等）。 Rapid7 提供互動式儀表板，能輕鬆將情報轉化為行動、管理警報、簡化操作並降低風險。透過一鍵修復，您可以直接在警報中達到跨團隊合作，落實威脅修復的生命週期。

將情資轉化為行動

您將了解每種特定的威脅，如 IOC、CVE、洩露的數據庫及黑客互動將如何影響您的業務。Rapid7 Threat Command 是市場上唯一的 All-In-One 解決方案，旨在應對網絡威脅各階段的挑戰， 從攻擊指示到妥協，再到調查和補救。

特點和能力

Rapid7 獨特的搜尋引擎，可從網路上最深和最難到達的地方收集情報，以識別您在其他情況下可能偵測不到的威脅，Rapid7 將寶貴的網路威脅偵測轉化為可操作的安全洞察力，提供以下功能：

  • 資產基礎上的智慧：過濾雜訊並專注於相關問題。使用您獨特的品牌和數位足跡資產，Threat Command平台可快速識別與脈絡化，針對即將發生的攻擊發出即時警訊。
  • 一鍵式修復：面對威脅，簡化後續的反應流程。立即將可疑網域發送到阻止列表（黑名單），快速啟動威脅清除，在不斷收到補救狀態通知的同時，持續追蹤惡意活動直到威脅清除。
  • 專業分析師協助：直接與Rapid7情報專家互動，網路分析師具有獨特的「由外而內」視角並了解威脅參與者的思考和運作方式，以獲取更多背景資訊並啟動更深入的資安鑑識與調查。
  • 多維度威脅分析：隨著時間推移，監控和驗證威脅以了解完整的威脅脈絡。威脅命令將與特定威脅相關的多個輸入、變量和交互關聯起來。
  • 隱蔽式數據偵察：識別所有威脅，無論其來源如何，不引起威脅對您的關注或使您企業處於危險之中。
  • 警報分析器：透過機器學習/人工智慧，快速確認威脅狀態。提供微調警報確認，進而讓 SOC 團隊花費更少的時間來整理不相關的警報。

案例分享- 信用卡發卡商利用 Threat Command 進行品牌保護

某信用卡發卡商痛點
  • 擁有超過 500 萬名客戶和 210 個店面。超過 400 萬持卡人和 900 萬張活躍卡，每年進行 1.46 億筆交易，旗下有一款行動 App。
  • 資安團隊缺乏對外部威脅的可見性。
  • 公司的線上品牌保護困難，如：惡意欺騙網域、機敏資料洩漏、假冒 App 等。
Threat Command 解決方案
  • Step1 - 主動識別外部網路威脅，將外部情報對應到企業獨特的數位資產，從明網、深網和暗網中，發現針對企業的關鍵威脅。
  • Step2 - 提供量身定制的安全情報，依據嚴重性及風險類型自動提供警報。
  • Step3 - 了解公司的數位足跡和人員的威脅，包含：信用卡號碼洩漏、欺騙性域名、針對企業高管和 VIP 的威脅後，直接在 Threat Command 平台上進行威脅緩解與下架。
